# JavaScript Challenges - Udemy

- Repositório criado para armazenar exercícios e desafios resolvidos durante meus estudos de JavaScript na Udemy.

## Objetivo

Este repositório tem como finalidade:

- Praticar lógica de programação;
- Consolidar conceitos de JavaScript;
- Registrar minha evolução nos estudos;
- Criar um histórico de soluções e aprendizados.

## Conteúdo

Os desafios abordam temas como:

- Variáveis e tipos de dados
- Operadores
- Estruturas condicionais
- Laços de repetição
- Funções
- Arrays
- Objetos
- ES6+
- Algoritmos e lógica de programação

## Estrutura

```text
desafios/
├── 01-variaveis/
├── 02-operadores/
├── 03-condicionais/
├── 04-loops/
└── ...
```

Cada pasta contém a solução do desafio correspondente e, quando necessário, explicações sobre a lógica utilizada.

## Tecnologias

- JavaScript (ES6+)
- Node.js
- Git
- GitHub

🚀 Em andamento — novos desafios serão adicionados conforme avanço no curso.
